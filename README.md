# Daily Security Briefing

An automated, **personal** open-source situation briefing covering four standing areas —
Domestic US Security, China / Taiwan, Russia / Ukraine, and US / Iran — published each
morning under `briefings/YYYY-MM-DD.md`.

## Purpose

This repository is a **personal situational-awareness feed** and a working demonstration of
an entirely **local** OSINT summarization pipeline — no cloud LLM; all inference runs on
hardware I control. It is *not* a professional intelligence product, a threat feed, or a
commercial deliverable.

## How each briefing is produced

The pipeline runs unattended each morning on a Raspberry Pi:

1. **Gather** — fetch recent items from a curated set of recognized OSINT and defense/analysis
   outlets (listed below) via their public RSS feeds.
2. **Summarize** — a **local** language model (gpt-oss-20b, run via LM Studio on my own
   machine) writes a short prose summary of each section **from the fetched headlines and
   excerpts only.**
3. **Cite** — every source link is taken **directly from the source feed**. URLs are not
   generated by the model.
4. **Publish** — the assembled markdown is committed here and delivered to a private channel.

## Sourcing

Section summaries are drawn from recent reporting/analysis by recognized outlets, including:

- **US homeland & defense:** CISA, Defense One, FDD, Long War Journal, War on the Rocks
- **China / Taiwan:** The Diplomat, Jamestown Foundation, Taipei Times
- **Russia / Ukraine:** Bellingcat, Oryx, Ukrainska Pravda (English), The Moscow Times
- **US / Iran:** FDD, Long War Journal, Times of Israel

Each section also lists recognized **OSINT X (Twitter) accounts** as a follow-reference for
that topic — these are pointers for the reader, not scraped sources.

> Older dates (2026-06-19 → 2026-06-29) were **backfilled** from date-ranged archive search and
> are lower-confidence than the live daily runs.

## ⚠️ Important — read before relying on anything here

- These briefings are **machine-generated summaries of secondary reporting.** They are **not
  verified intelligence, not professional analysis, and not advice.**
- The summarizing model **can and occasionally will err** — misattribute an event, garble a
  figure, or omit context. **Always verify against the cited primary sources before acting on
  or citing anything here.**
- Content reflects **what the fetched sources reported**, not independent confirmation of truth.
  Some monitored outlets are state-affiliated and are included for coverage, not endorsement.
- Nothing here represents the views of any employer, client, or affiliated organization.
  Provided **as-is**, for personal awareness only.

## Structure

Each `briefings/YYYY-MM-DD.md` contains four sections; each section has a narrative summary,
the source links used, and a list of relevant OSINT accounts.
